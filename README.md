# PromptPay-QR-Code-Generator
 
** Clone โปรเจกต์จาก Git **
git clone https://github.com/Tadorupakinson/PromptPay-QR-Code-Generator.git

** เข้าไปภายในไฟล์ **
cd PromptPay-QR-Code-Generator\promptpay-app

** ติดตั้ง dependencies **
npm install

** ใช้งานโปรแกรม **
npm run dev

** เปิดเว็บไซต์ **
http://localhost:3000
